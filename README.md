# I coded this to use jsonbin but you can of course change it to what you would like to use.
# I also used Oracle Cloud VPS for my server.
Make sure to save your jsonbin id as a "JSON_BIN_ID" env variable on your server.
Make sure to save your jsonbin master api key as a "BIN_API_KEY" env variable on your server.
Make sure to save your server ip as a "server_ip" env variable on your local pc.
# You can always hard code these; however, if you will be sharing these files I do not recommend it.
